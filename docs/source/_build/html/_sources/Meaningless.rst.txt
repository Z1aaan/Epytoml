What is Meaningless?
--------------------
| Meaningless is a dummy text generator for Epytoml and it's modules.
| Can be used in Notaker and other Epytoml modules.

Using Meaningless
------------------

| Importing Meaningless

.. code-block:: python

    from Epytoml import Meaningless
